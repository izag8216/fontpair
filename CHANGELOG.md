# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-04-20

### Added
- Initial release
- Font scanning for macOS, Linux, Windows (OTF/TTF/TTC)
- Auto-categorization (serif, sans-serif, monospace, display, handwriting)
- Font metric analysis (x-height ratio, weight, contrast, avg width)
- Pairing recommendation engine with 6 style presets
- Scoring system (visual harmony, readability, contrast, x-height compat)
- Export to CSS, JSON, YAML formats
- SQLite cache for font metadata
- Rich terminal output with tables and panels
- CLI with scan, recommend, export, info commands
